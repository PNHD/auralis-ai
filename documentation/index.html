Auralis AI Documentation

Version 1.0.0

Auralis AI is a ThemeForest-ready Next.js + Tailwind CSS + TypeScript SaaS and Admin Dashboard Multipurpose Template. Featuring 184+ high-fidelity routes, 12 dashboard layouts, dynamic RTL/LTR support, and Light/Dark modes.

Key Features

  • Next.js (App Router): Modern static routing with state-of-the-art architectures.
  • Tailwind CSS: Customized theme structure using Tailwind inline theme imports.
  • RTL support: Fully responsive direction toggles for Arabic and Hebrew support.
  • Light & Dark modes: Smooth class-based theme switcher saved in local storage.
  • Dynamic routing: 184+ pages structured dynamically via static parameters, ready to be linked to APIs.
  • No jQuery: 100% clean component-based React framework.

Setup & Installation

Ensure you have Node.js installed on your local machine, then open your terminal inside the project directory and run:

# Install node modules
npm install

# Start local dev server
npm run dev

# Compile static HTML files for hosting
npm run build

Static build files will compile into the dist/ or out/ directory, ready to deploy to Cloudflare Pages or Vercel.

Routing & Pages

All template pages are defined inside src/data/pagesDb.ts. You can easily add, rename, or structure routes without writing new JSX files. To map a path to your own custom page design, modify src/components/PageRenderer.tsx.

Theme & RTL Customization

Theme colors and variables are configured inside src/app/globals.css. Layout direction and dark/light switching states are managed dynamically via the client context wrapper inside src/components/Providers.tsx.
