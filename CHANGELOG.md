# Changelog - Auralis AI Next.js Template

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-03
### Added
- Initial Release of Auralis AI Multi-Purpose Next.js + Tailwind CSS SaaS & Admin Dashboard Template.
- 184+ routes scaffolding across 13 core categories (Dashboards, AI Apps, CRM, eCommerce, LMS, Healthcare, HR, Finance, etc.).
- 12 Dashboard layouts including AI SaaS, Analytics, CRM, Project, LMS, eCommerce, Healthcare, HR, Marketing, Finance, Operations.
- RTL layout direction switcher support (LTR/RTL toggles).
- Dynamic Light and Dark modes switcher utilizing localStorage preferences.
- Standard ThemeForest-grade layout components: Navbar, Sidebar drawers, Recharts adapters.
- Fully static export configs ready for Cloudflare Pages and Vercel.
- Integrated styling design guidelines.
